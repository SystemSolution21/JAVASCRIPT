<h2>Javascript to read the content of a text file and display when the webpage loads.</h2>
<h3>Features:</h3>
<ol>
    <li>element with the id "content" where the text display.</li>
    <li>The window onload event ensures the code runs after the page has fully loaded.</li>
    <li>fetch API to read the content of a file named 'content.txt'.</li>
    <li>Once the file is read, set the text content to element InnerHtml.</li>
    <li>If there's an error (e.g., file not found), catch it and display an error message.</li>
</ol>
<h3>To use this:</h3>
<ol>
    <li>Create index.html file</li>
    <li>Create a text file named `content.txt` in the same directory.</li>
    <li>Put the text to display in `content.txt`.</li>
    <li>Create 'web-api.js' for function to read text file using 'fetch' API</li>
    <li>Open the HTML file in a web browser.</li>
</ol>
